# Apollo

A mopidy player with snapcast support

## Installation

Install by running:

```
python3 -m pip install Mopidy-Apollo
```

Before starting Mopidy, you must add configuration for Mopidy-Apollo to your Mopidy configuration file:

```
[apollo]
enabled = true
```
